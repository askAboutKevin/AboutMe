Projects

Cash Register

For this project I wanted to create a system that would produce the same functionality as a cash register in a supermarket. During the process of creating the project I added more features so that it could function as a RESTful API. Using the Dropwizard framework, MySQL database, JDBI3 library, HTML, CSS and JavaScript I was able to create an API that could accept HTTP requests and interact with a local database to create, read, update and delete data so that users could complete orders from the the web interface.

Tortoise and Hare

One of my first coding assignments at school. Using C++ and object-oriented programming techniques I was able to create a program that could simulate a race between a Tortoise and Hare character based on the well-known folktale.
